# 楽譜を書く

## 楽譜の記述方法

- [alphaTex](https://www.alphatab.net/docs/alphatex/introduction)

  - ベースはこの書き方を採用しています

- [読み込み可能なファイル](https://www.alphatab.net/docs/introduction#the-file-importers)で楽譜を書き、読み込む

## ORIGINAL

ORIGINAL の場合は、自分でスコアを編集することができます。

## COPY

COPY の場合は、管理者が誰もいないので差分管理を行う必要があります。
プログラマーであれば Github などをイメージしていただくとわかりやすいです。

1. Pull を作成する

   Pull を作成すると、作成した時点のスコアの状態をコピーし、自由に編集することができます。

2. Pull にスコアを書く

   Pull では新しく楽譜を追加したり、間違いを修正することができます。

3. PullRequet を公開する

   - 貢献者が少数の場合自身でマージすることができる。
   - 貢献者が複数いる場合投票を行う。
     貢献者の人数によって自動的に投票期間が決まり、
     賛成票が多ければマージされ、反対票が多ければドラフト状態にされる。

4. Pull をマージする

   Pull をマージすると、コピー元の楽譜に Pull の変更が反映されます。
