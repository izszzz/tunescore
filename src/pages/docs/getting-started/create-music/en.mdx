# Create Score

1. access[create page](/new)

2. fill in the required fields on the form

| item        | description                            |
| ----------- | -------------------------------------- |
| type        | type of data to be created             |
| name        | name of data                           |
| copyright   | copyrighted or original                |
| visibillity | whether the data is publicly available |

3. file loading

   If you have an image or file of the score, you can load it to read the initial data of the score.

- [readable files](https://www.alphatab.net/docs/introduction#the-file-importers)
- readable images `.jpg` `.png` `.pdf`

4. submission

   Please fill in the required fields and then submit.
