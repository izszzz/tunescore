# tunescore とは

tunescore とは楽譜に特化した音楽情報サイトです。

## 目的

- 著作権フリーの楽譜や音楽をアーカイブする
- アーティストやミュージシャンの収益化
- 音楽に関する情報の整理

## 機能

- 差分形式での楽譜管理
- 楽譜の販売/閲覧
- 音楽にかかわる情報の整理

## ロードマップ

### 貢献者への報酬

- 楽譜の修正や情報整理につきポイントを配布し、換金できるようにします。
- このサービスは Github の OSS として公開されているので、プルリクエストに報酬を与える予定です。

### 著作物の管理

- 著作物の楽譜も管理できるように、著作管理団体と包括契約を結びます。

### 開発者向け API の解放

- 蓄積された楽譜や情報は、今後の音楽サービスやコンテンツの向上につながります。

### 既存の楽譜販売業者との協力

- 楽譜の販売は様々な企業が行っていますが、採譜者のミスは多々あります。
  そのような楽譜でもアップロードしていただくことで、ミスを誰かが修正しバージョン管理をすることが可能になります。

### 信頼度ランク

これはユーザーが増えてからですが、悪質なユーザーに情報を破壊されないために編集権限をランクごとに設定します

## 寄付
