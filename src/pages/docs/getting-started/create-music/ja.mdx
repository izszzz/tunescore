# 楽譜の作成

1. [作成ページ](/new)にアクセスする

2. フォームに必須項目を入力する

|    項目     |         説明         |
| :---------: | :------------------: |
|    type     | 作成するデータの種類 |
|    name     |     データの名前     |
|  copyright  |  著作物かオリジナル  |
| visibillity |  データを公開するか  |

3. ファイル読み込み

   楽譜の画像やファイルを持っている場合それを読み込むことで、楽譜の初期データを読み込むことができます。

- [読み込み可能なファイル](https://www.alphatab.net/docs/introduction#the-file-importers)
- 読み込み可能な画像 `.jpg` `.png` `.pdf`

4. 提出

   必要な項目を埋めたのち提出をしてください。
